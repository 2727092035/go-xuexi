# Day 1 笔记：变量与类型

## 核心知识点

### 1. 变量声明三种方式

| 方式 | 语法 | 适用场景 |
|------|------|----------|
| 短声明 | `name := "Tom"` | 函数内部，最常用 |
| var 显式 | `var name string = "Tom"` | 函数外部（包级变量） |
| var 零值 | `var name string` | 需要零值初始化时 |

### 2. 基本类型 & 零值

| 类型 | 零值 | JS 对比 |
|------|------|---------|
| int | 0 | — |
| float64 | 0 | — |
| string | ""（空串） | JS 是 undefined |
| bool | false | JS 是 undefined |

### 3. 和 JS/TS 的关键差异

- **无隐式类型转换** — `int + float64` 直接报错，必须显式转
- **无 undefined/null** — 每种类型都有确定的零值
- **常量必须编译期确定** — 不能把函数返回值赋给 const
- **未使用的变量会编译报错** — JS 只是 warning，Go 直接不让编译

### 4. fmt 格式化占位符

- `%s` 字符串
- `%d` 整数
- `%f` 浮点（`%.2f` 保留两位小数）
- `%v` 任意值
- `%T` 打印类型
